# QQplug
 android QQ的插件 可备份/删除缓存(含闪照)

demo的apk(未签名)在 app/build/outputs/apk/debug/app-debug.apk

## 用法：

1.在主菜单点击功能

2.在新页面底部点击“使用此文件夹”按钮进行授权
